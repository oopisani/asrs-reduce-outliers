Code implementation for the case study on applying the `reduce_outliers()` method from BERTopic to aviation incident reports (ASRS 2010–2020).

## Overview
This repository contains the raw source code used to investigate thematic consistency and noise reduction in ASRS aviation reports (2010–2020). **The analysis focuses specifically on the optimization and thematic recovery of Topic 6 (smoke and fumes-related incidents).**

## Technologies
- Python 3.x
- BERTopic
- UMAP
- HDBSCAN
- Pandas / Scikit-learn
  
## Dataset
The dataset used in this study is sourced from the NASA Aviation Safety Reporting System (ASRS). You can access and download the raw data from the official portal:
[ASRS Online Database](https://asrs.arc.nasa.gov/search/database.html)
